About Oker

Oker is a programming language designed for readability and simplicity, featuring natural language keywords and intuitive syntax.

Features

  • Natural language keywords
  • Type inference
  • Built-in functions
  • Error handling
  • File I/O operations

Quick Reference

  • let x = 5 - Variable declaration
  • say "Hello" - Print output
  • makef func(): ... end - Function definition
  • if condition: ... end - Conditional
  • while condition: ... end - Loop
